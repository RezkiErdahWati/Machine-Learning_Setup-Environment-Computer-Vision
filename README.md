# Setup Environment Computer Vision

## Tujuan
Untuk melakukan instalasi dan melakukan konfigurasi environment untuk melakukan praktikum Machine Learning yang dikhususkan untuk Computer Vision,

##Tool yang Digunakan
- Anaconda
- Python 3.10.0
- JupyterLab

## Langkah - Langkah
1. Mendowload Anaconda terlebih dahulu

2. Membuat environment baru:
   conda create -n mlcv python=3.10

3. Mengaktifkan environment:
   conda activate mlcv

4. Install Library:
   pip install numpy matplotlib opencv-python

5. Instal JupyterLab:
   pip install JupyterLab

6. Menjalankan JupyterLab:
   jupyter Lab

## Hasil
Environment berhasil dibuat dan JupyterLab dapat di jalankan dengan kernel python aktif
